Setting Up

Add this to your code. Remember: (css: header | scripts: end of the body)

<link rel="stylesheet" href="path/to/easy-parallax.css">
<script type="text/javascript" src="path/to/easy-parallax.js">

The parallax will be in the div.

<div id="easy-parallax">your content goes here</div>

Place this in your js file.

$("div#easy-parallax section").startParallax(dataspeed); //it's 2 by default

Change the path on easy-parallax.css

background: url(path/to/img) 50% 0 fixed;
